Deskripsi Konteks

Outing class kali ini menggabungkan eksplorasi budaya dan matematika di ruang rias Anjungan Jawa Timur. Sambil memahami tata ruang budaya, kita juga belajar menghitung luas ruangan.

Masalah Matematika

Ruang rias berbentuk persegi panjang seluas 12 m², dengan panjang 1 meter lebih besar dari lebarnya. Berapa panjang dan lebar ruangan tersebut?

Pemodelan

Misalkan lebar ruangan = x, maka panjang = x + 1

L = p × l = x(x + 1) = 12
x² + x - 12 = 0

Penyelesaian

a. Metode Faktorisasi

Langkah 1: Tentukan persamaan kuadrat:

x² + x - 12 = 0

Langkah 2: Cari dua angka yang hasil kalinya -12 dan jumlahnya 1 (koefisien x).

Kita dapat: 4 × (-3) = -12 dan 4 + (-3) = 1

Langkah 3: Faktorkan:

(x + 4)(x - 3) = 0

Langkah 4: Selesaikan:

x = -4 atau x = 3

Karena ukuran tidak mungkin negatif, maka x = 3

Lebar = 3 m, Panjang = 4 m

b. Metode Rumus Kuadrat (ABC)

Persamaan: x² + x - 12 = 0, maka:

a = 1, b = 1, c = -12

Gunakan rumus:

x = (-b ± √(b² - 4ac)) / 2a

Hitung:

x = (-1 ± √(1² + 48)) / 2 = (-1 ± √49)/2
x = (-1 ± 7)/2

Hasil:

x = 3 atau x = -4

Lebar = 3 m, Panjang = 4 m

c. Metode Melengkapkan Kuadrat

Langkah 1: Pindahkan -12 ke kanan:

x² + x = 12

Langkah 2: Lengkapi kuadrat:

Ambil ½ dari koefisien x: (½) = 0.5, kuadratkan = 0.25

Langkah 3: Tambahkan ke kedua sisi:

x² + x + 0.25 = 12 + 0.25
(x + 0.5)² = 12.25

Langkah 4: Akar kuadrat kedua sisi:

x + 0.5 = ±√12.25 = ±3.5

Hasil:

x = 3 atau x = -4

Lebar = 3 m, Panjang = 4 m

d. Analisis Budaya

Ruang rias bukan hanya tempat bersiap, tapi juga ruang transisi sakral. Ukurannya dirancang dengan filosofi harmoni, mencerminkan keteraturan budaya Jawa. Melalui perhitungan luas, kita belajar menghargai ruang secara matematis dan kultural.

e. Kesimpulan

Matematika tidak hanya menyelesaikan angka, tetapi juga membantu memahami nilai dan keteraturan dalam budaya. Outing class ini menunjukkan bahwa budaya dan matematika berjalan bersama menciptakan harmoni dalam kehidupan.
